# 🏫 Система управления онлайн-школой v2.0

Расширенная версия с модулями, уроками, домашними заданиями и визуализацией.

## 📋 Новые функции

### Задание 6: Модули и уроки
- Курс → Модули → Уроки
- Добавление модулей и уроков через меню

### Задание 7: Домашние задания
- ДЗ к каждому уроку
- Выставление оценок за ДЗ
- Учёт ДЗ в итоговой оценке

### Задание 8: Аналитика по преподавателю
- Количество активных курсов
- Средняя успеваемость студентов
- Список завершённых курсов с датами

### Задание 9: Экспорт в CSV
- Отчёт по студенту
- Отчёт по преподавателю
- Отчёт по курсу

### Задание 10: Визуализация
- Графики успеваемости студента
- Распределение оценок
- Оценки за домашние задания
